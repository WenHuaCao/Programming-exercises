# Week1 [Basic C++] default arguments

## Description
```
写一个sum函数,体会函数缺省值的使用.
 
对于主程序
int main ( ) 
{
cout << sum() << endl;
cout << sum(6) << endl;
cout << sum(6, 10) << endl;
cout << sum(6, 10, 20) << endl;
return 0;
}
有如下的输出
```
## Sample Output
```
100
106
16
36
```
