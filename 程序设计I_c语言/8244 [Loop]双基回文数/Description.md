# [Loop]双基回文数

# Description
如果一个正整数n至少在两个不同的进位制b1、b2下都是回文数（2 <= b1, b2 <= 10），则称n为双基回文数（注意：回文数不能包含前导0）。输入非负整数S < 10^6，输出比S大的最小双基回文数。

【输入数据第一行是样例个数T（1<=T<=50），接下来有T行，每行有一个非负整数S（小于10 ^6）】

【输出数据有T行，每行输出对应的最小双基回文数】

# Sample Input
```
3
555555
231231
123456

```

# Sample Output
```
563786
232263
124670

```
